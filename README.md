# try-astro-hono-clerk

Astro, Hono, Clerkを使ってみる試み

# setup command

```bash
% touch .mise.toml
% mise trust -a
% cat .mise.toml
[tools]
node = "24.8.0"

% mise install
# npm version up
% npm install -g npm@11.7.0

# 以下のコマンドが良さそう
% npm create astro@latest astro -- --template minimal --install --no-git --yes

```
